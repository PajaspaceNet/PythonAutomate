## Porovnej 

Tento nástroj umožňuje jednoduše porovnat, zda soubor B obsahuje všechny IP adresy uvedené v souboru A.</br>
IP adresy mohou být součástí libovolného textu a mohou obsahovat i masky typu /32, které jsou automaticky ignorovány.</br>
Stačí vložit obsah obou souborů do HTML formuláře nebo spustit Python skript a nástroj vyhodnotí chybějící IP.
Výsledek se zobrazí přehledně na obrazovce a lze jej snadno zkopírovat či uložit.
Projekt slouží jako rychlá pomůcka pro validaci síťových konfigurací či audit IP seznamů
